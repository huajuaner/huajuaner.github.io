queues

  • 传值
  • 通过传递指针同样可以实现传指针（多个tasks 共享一个内存空间）
  • 由内核负责queue的存储空间分配
  • 在中断中使用的api不同于task中
  • 对queue 的操作如果不合法，会导致当前的task blocked

Note that interrupts must NOT use API functions that do not end in "FromISR".
